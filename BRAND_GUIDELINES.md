## üìù Typography

### Font Families

#### Primary Font: System Default
- **iOS:** San Francisco
- **Android:** Roboto
- **Web:** System UI Font Stack

#### Display Font (for headings)
- **Serif:** Georgia, Times New Roman (for elegant headers)
- **Sans-serif:** System default (for modern headers)

### Type Scale
- **H1:** 36px, Bold, Purple (#4A2663)
- **H2:** 30px, Bold, Purple (#4A2663)
- **H3:** 24px, Semibold, Purple (#4A2663)
- **H4:** 20px, Semibold, Purple (#4A2663)
- **H5:** 18px, Medium
- **H6:** 16px, Medium
- **Body 1:** 16px, Regular
- **Body 2:** 14px, Regular
- **Caption:** 12px, Regular, Gray (#757575)
- **Button:** 16px, Semibold, Uppercase

### Typography Guidelines
- **Line Height:** 1.5 for body text, 1.2 for headings
- **Letter Spacing:** Default for body, 0.5px for buttons
- **Paragraph Spacing:** 16px between paragraphs
- **Max Line Length:** 70 characters for optimal readability

## üìê Spacing & Layout

### Grid System
- **Base Unit:** 8px
- **Padding:** 16px (mobile), 24px (tablet), 32px (desktop)
- **Margins:** Multiples of 8px (8, 16, 24, 32, 48, 64)
- **Column Gutters:** 16px

### Border Radius
- **Small:** 4px (buttons, inputs)
- **Medium:** 8px (cards)
- **Large:** 12px (modals)
- **Extra Large:** 16px (hero sections)
- **Full:** 9999px (pills, avatars)

### Shadows
- **Small:** `0 1px 3px rgba(74, 38, 99, 0.12)`
- **Medium:** `0 4px 6px rgba(74, 38, 99, 0.16)`
- **Large:** `0 10px 20px rgba(74, 38, 99, 0.20)`
